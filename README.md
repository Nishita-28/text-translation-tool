# 🌍 Text Translation Tool

A Python tool that extracts and translates text from files or websites.

---

## 🚀 Features

* Translate text from files
* Extract text from websites
* Supports multiple languages
* Saves translated output

---

## 🛠 Tech Stack

* Python
* requests
* BeautifulSoup
* googletrans

---

## ▶️ How to Run

```bash
python main.py
```
